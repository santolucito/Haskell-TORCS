\begin{abstract}
Functional Languages have had major benefits to the programming community.
Although features such as purity, a strong type system, can help guide programmers away from costly errors, these present challenges when used in a reactive system.
Functional Reactive Programming is a paradigm that allows users the benefits of functional languages and an easy interface to a reactive environment.
We present a tool for building autonomous vehicle controllers in FRP using Haskell.
\end{abstract}
